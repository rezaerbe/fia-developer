## Female In Action

Female In Action is application about women's empowerment in the form of articles, videos, report, coaching, and consultation.

## Tools

* Android Studio
* Java Language
* Firebase

## Dependencies

* Material
* Firebase Auth
* Firebase Firestore
* Glide
* Butterknife
* Paging
* Lifecycle
